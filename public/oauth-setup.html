Step 4: Manual Token Exchange (if needed)

If the automatic exchange doesn't work, you can manually exchange the authorization code:

Manual curl command:
curl -X POST https://accounts.zoho.com/oauth/v2/token \
  -d "code=AUTHORIZATION_CODE" \
  -d "client_id=YOUR_CLIENT_ID" \
  -d "client_secret=YOUR_CLIENT_SECRET" \
  -d "redirect_uri=https://dc-commissions.vercel.app/oauth-callback.html" \
  -d "grant_type=authorization_code"
